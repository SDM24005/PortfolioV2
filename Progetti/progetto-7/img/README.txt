Cartella per i file media del progetto

Inserisci qui:
- preview.jpg (OBBLIGATORIA - immagine per la galleria)
- Le altre immagini del progetto (.jpg, .png)
- Video (.mp4, .webm)
  - Per video loop: aggiungi -loop al nome (es: video-loop.mp4)
- PDF (.pdf)

IMPORTANTE:
- Il nome del file preview DEVE essere esattamente "preview.jpg"
- Evita spazi nei nomi dei file (usa - o _ invece)
- Usa solo lettere, numeri, trattini e underscore
